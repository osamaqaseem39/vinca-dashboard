## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The dashboard will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Design System

- **Font**: Poppins (weights: 300, 400, 500, 600, 700)
- **Color Scheme**: White background with black borders
- **Border Radius**: Rounded corners (xl, 2xl)
- **Typography**: Elegant font weights for hierarchy
